# 🚀 Beginner CI/CD Pipeline Project

This project demonstrates a simple Continuous Integration (CI) pipeline using GitHub Actions and GitHub Pages.

The workflow automatically runs whenever code is pushed to the `main` branch.

---

# 📌 Project Features

- ✅ GitHub Actions Workflow
- ✅ Automated CI Pipeline
- ✅ HTML Static Website
- ✅ GitHub Pages Deployment
- ✅ Automatic Build Trigger on Push

---

# 🛠️ Technologies Used

- HTML5
- GitHub Actions
- GitHub Pages
- YAML
